# Atlassian Jira Scripts
Powershell scripts for interacting with Atlassian Jira with the [Aberrant](https://www.aberrant.io/) RemoteAgent

## Contents

| Script| Description |
| -------- | ------- |
| create_jira_ticket.ps1  | Creates a ticket in Jira with given name and description. Requires configuration of Email Address (ticket owner), API Token, Jira project and Url. |
